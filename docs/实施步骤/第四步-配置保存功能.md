## 1. 实现新增文件夹功能

### 1.1 更新 Extension 消息处理

在 `src/extension.ts` 中添加 `addFolder` 消息处理：

```typescript
case 'addFolder':
  const folderName = await vscode.window.showInputBox({
    prompt: '请输入文件夹名称',
    placeHolder: '例如: code, libs, docs',
    validateInput: (value) => {
      if (!value || value.trim() === '') {
        return '文件夹名称不能为空';
      }
      if (value.includes('/') || value.includes('\\')) {
        return '文件夹名称不能包含路径分隔符';
      }
      return null;
    }
  });

  if (folderName) {
    publishFlowManager?.addFolder(folderName.trim());
    await publishFlowManager?.save();
    
    panel.webview.postMessage({
      type: 'configUpdated',
      payload: publishFlowManager?.getConfig()
    });
    
    vscode.window.showInformationMessage(`已创建文件夹: ${folderName}`);
  }
  break;
```

## 2. 实现文件选择对话框

### 2.1 更新 Extension 消息处理

在 `src/extension.ts` 中添加 `selectFiles` 消息处理：

```typescript
case 'selectFiles':
  const selectedFiles = await vscode.window.showOpenDialog({
    canSelectFiles: true,
    canSelectFolders: true,
    canSelectMany: true,
    openLabel: '添加到发布清单',
    defaultUri: workspaceFolder.uri
  });

  if (selectedFiles && selectedFiles.length > 0) {
    for (const file of selectedFiles) {
      const filePath = file.fsPath;
      const stats = fs.statSync(filePath);
      
      if (stats.isDirectory()) {
        // 添加整个文件夹
        await addFolderRecursiveToTarget(
          filePath,
          publishFlowManager!,
          undefined
        );
      } else {
        // 添加单个文件
        const targetPath = path.basename(filePath);
        publishFlowManager?.addFile(filePath, targetPath);
      }
    }
    
    await publishFlowManager?.save();
    
    panel.webview.postMessage({
      type: 'configUpdated',
      payload: publishFlowManager?.getConfig()
    });
    
    vscode.window.showInformationMessage(`已添加 ${selectedFiles.length} 个文件/文件夹`);
  }
  break;
```

## 3. 完善保存配置功能

### 3.1 更新 Extension 消息处理

在 `src/extension.ts` 中完善 `saveConfig` 消息处理：

```typescript
case 'saveConfig':
  try {
    await publishFlowManager?.save();
    
    // 显示保存成功通知
    const config = publishFlowManager?.getConfig();
    const { totalFiles, totalFolders, totalSize } = config?.metadata || {};
    
    vscode.window.showInformationMessage(
      `配置已保存: ${totalFiles} 个文件, ${totalFolders} 个文件夹`,
      '查看配置'
    ).then(selection => {
      if (selection === '查看配置') {
        const configPath = path.join(
          workspaceFolder.uri.fsPath,
          '.releasePlan',
          'publish-config.json'
        );
        vscode.workspace.openTextDocument(configPath).then(doc => {
          vscode.window.showTextDocument(doc);
        });
      }
    });
  } catch (error) {
    vscode.window.showErrorMessage(`保存配置失败: ${error}`);
    panel.webview.postMessage({
      type: 'error',
      payload: { message: String(error) }
    });
  }
  break;
```

### 5.2 添加全局错误边界

创建 `webview/src/components/ErrorBoundary.vue`：

```vue
<template>
  <div v-if="hasError" class="error-boundary">
    <div class="error-content">
      <span class="codicon codicon-error"></span>
      <h3>发生错误</h3>
      <p>{{ errorMessage }}</p>
      <button @click="resetError">重试</button>
    </div>
  </div>
  <slot v-else></slot>
</template>

<script setup lang="ts">
import { ref, onErrorCaptured } from 'vue';

const hasError = ref(false);
const errorMessage = ref('');

onErrorCaptured((err) => {
  hasError.value = true;
  errorMessage.value = err.message || '未知错误';
  console.error('Error captured:', err);
  return false;
});

const resetError = () => {
  hasError.value = false;
  errorMessage.value = '';
  window.location.reload();
};
</script>

<style scoped>
```

## 7. 常见问题排查

### 问题 1: 配置文件未生成
**排查步骤**:
1. 检查工作区是否有写权限
2. 检查 `.releasePlan` 目录是否创建
3. 查看 Debug Console 错误日志

### 问题 2: 保存后数据丢失
**排查步骤**:
1. 检查 `PublishFlowManager.save()` 是否正确调用
2. 检查 JSON 序列化是否成功
3. 检查文件写入是否成功

### 问题 3: 删除按钮不显示
**排查步骤**:
1. 检查 `showDelete` prop 是否正确传递
2. 检查 CSS 样式是否正确
3. 检查事件绑定是否正确
