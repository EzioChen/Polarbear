# 发布流程配置方案设计文档

## 1. 功能概述

设计并实现一个发布流程配置功能，允许用户通过图形界面选择需要发布的文件和文件夹，并将配置保存为本地JSON文件。

## 2. 详细需求

### 2.1 入口设计

在现有系统中实现"增加发布流程"选项，用户选中该选项后应导航至新创建的选配View界面。

**实现方式：**
- 点击侧边栏"增加发布流程"按钮
- 打开一个新的 Webview Panel，标题为"配置发布流程"
- Webview 使用 HTML + CSS + JavaScript 构建双面板界面

### 2.2 选配View界面组件

#### 2.2.2 界面组件详细说明

**顶部工具栏：**
| 按钮 | 图标 | 功能描述 |
|------|------|----------|
| 选择发布文件/文件夹 | 📂 | 打开系统文件选择对话框，支持多选 |
| 新增文件夹 | 📁+ | 在右侧面板创建新的文件夹节点 |
| 保存配置 | 💾 | 将当前配置保存到 `.releasePlan` 目录 |

**左侧面板（项目文件）：**
- 显示当前工作区文件夹的完整文件树
- 支持展开/折叠文件夹
- 支持点击选中文件/文件夹
- 显示文件图标（根据文件类型）
- **支持拖拽文件/文件夹到右侧面板**
- 右键菜单：添加到发布清单

**右侧面板（发布清单）：**
- 显示已选择的文件和文件夹
- **支持从左侧面板接收拖拽的文件/文件夹**
- **支持内部拖拽调整层级关系（文件可拖入文件夹）**
- 支持重命名文件夹
- 支持删除节点
- 显示文件层级结构

#### 2.2.3 拖拽交互设计

**拖拽场景说明：**

| 拖拽方向 | 源 | 目标 | 操作结果 |
|---------|-----|------|----------|
| 左 → 右 | 项目文件 | 发布清单根目录 | 复制文件/文件夹到发布清单 |
| 左 → 右 | 项目文件 | 发布清单文件夹 | 复制文件到指定文件夹内 |
| 右 → 右 | 发布清单文件 | 发布清单文件夹 | 移动文件到文件夹内 |
| 右 → 右 | 发布清单文件夹 | 发布清单根目录 | 移动文件夹到根目录 |

**拖拽视觉反馈：**
- **拖拽开始时**：被拖拽项半透明化（opacity: 0.5），鼠标变为拖拽手势
- **拖拽过程中**：显示拖拽预览卡片，显示被拖拽项名称和图标
- **悬停可放置区域时**：目标区域高亮显示（边框变蓝/背景变色）
- **释放时**：目标区域闪烁一下表示成功接收

**拖拽约束：**
- 左侧面板内的项不可互相拖拽（只读）
- 右侧面板内的文件夹不能拖入自己的子文件夹（防止循环）
- 同名文件/文件夹拖入时提示是否覆盖或重命名

**底部状态栏：**
- 显示当前选择的统计信息
- 显示配置文件保存路径

### 2.3 文件层级构建

右侧面板需支持构建文件层级结构，例如：

```
code/
  ├─ file1.js
  └─ subfolder/
      └─ file2.js
```

**实现方式：**
- 使用树形组件展示层级结构
- 支持拖拽节点调整位置
- 支持右键菜单创建子文件夹
- 文件夹可以嵌套任意层级

## 3. 技术实现

### 3.1 UI 部分

#### 3.1.1 技术选型
- **Webview**: VS Code 提供的 Webview API
- **前端框架**: Vue 3（Composition API，更好的组件化和响应式管理）
- **构建工具**: Vite（快速构建，支持开发热更新）
- **组件库**: 自定义组件 + VS Code 的 CSS 变量保持风格一致
- **图标**: VS Code Codicons 图标库
- **拖拽库**: VueDraggable（基于 Sortable.js，支持 Vue 的拖拽排序）

**Vue 3 选型理由：**
1. **组件化开发**: 将文件树、拖拽区域、工具栏封装为独立组件，便于维护
2. **响应式数据**: 使用 `ref` 和 `reactive` 管理文件树状态，自动更新视图
3. **Composition API**: 逻辑复用，将拖拽逻辑、文件操作封装为可复用函数
4. **TypeScript 支持**: 更好的类型推断和代码提示
5. **生态丰富**: VueDraggable 等库提供成熟的拖拽解决方案

#### 3.1.3 Vue 组件实现

**App.vue - 根组件：**
```vue
<template>
  <div class="publish-flow-container">
    <Toolbar 
      @select-files="handleSelectFiles"
      @add-folder="handleAddFolder"
      @save-config="handleSaveConfig"
    />
    
    <div class="main-content">
      <ProjectPanel 
        :files="projectFiles"
        @drag-start="handleDragStart"
      />
      
      <PublishPanel 
        :files="publishFiles"
        @drop="handleDrop"
        @move-item="handleMoveItem"
        @delete-item="handleDeleteItem"
      />
    </div>
    
    <StatusBar :stats="publishStats" />
  </div>
</template>

<script setup lang="ts">
import { ref, computed, onMounted } from 'vue';
import Toolbar from './components/Toolbar.vue';
import ProjectPanel from './components/ProjectPanel.vue';
import PublishPanel from './components/PublishPanel.vue';
import StatusBar from './components/StatusBar.vue';
import type { FileNode, PublishStats } from './types';
import { useVscodeApi } from './composables/useVscodeApi';

const { postMessage, onMessage } = useVscodeApi();

const projectFiles = ref<FileNode[]>([]);
const publishFiles = ref<FileNode[]>([]);
const draggedItem = ref<FileNode | null>(null);
const draggedSource = ref<'left' | 'right' | null>(null);

const publishStats = computed<PublishStats>(() => {
  let totalFiles = 0;
  let totalFolders = 0;
  let totalSize = 0;
  
  const traverse = (nodes: FileNode[]) => {
    for (const node of nodes) {
      if (node.type === 'file') {
        totalFiles++;
        totalSize += node.size || 0;
      } else {
        totalFolders++;
        if (node.children) traverse(node.children);
      }
    }
  };
  
  traverse(publishFiles.value);
  return { totalFiles, totalFolders, totalSize };
});

onMounted(() => {
  // 请求初始数据
  postMessage({ type: 'getWorkspaceFiles' });
  postMessage({ type: 'getPublishConfig' });
  
  // 监听消息
  onMessage((message) => {
    switch (message.type) {
      case 'workspaceFiles':
        projectFiles.value = message.payload;
        break;
      case 'configUpdated':
        publishFiles.value = message.payload.files;
        break;
    }
  });
});

const handleDragStart = (item: FileNode, source: 'left' | 'right') => {
  draggedItem.value = item;
  draggedSource.value = source;
};

const handleDrop = (targetPath: string | null) => {
  if (!draggedItem.value) return;
  
  postMessage({
    type: draggedSource.value === 'left' ? 'addFromProject' : 'moveInPublishList',
    payload: {
      itemId: draggedItem.value.id,
      sourcePath: draggedItem.value.sourcePath,
      targetPath
    }
  });
  
  draggedItem.value = null;
  draggedSource.value = null;
};

const handleSelectFiles = () => {
  postMessage({ type: 'selectFiles' });
};

const handleAddFolder = () => {
  postMessage({ type: 'addFolder' });
};

const handleSaveConfig = () => {
  postMessage({ type: 'saveConfig' });
};

const handleMoveItem = (itemId: string, targetPath: string) => {
  postMessage({
    type: 'moveInPublishList',
    payload: { itemId, targetPath }
  });
};

const handleDeleteItem = (itemId: string) => {
  postMessage({
    type: 'deleteItem',
    payload: { itemId }
  });
};
</script>

.publish-flow-container {
  display: flex;
  flex-direction: column;
  height: 100vh;
  font-family: var(--vscode-font-family);
  color: var(--vscode-foreground);
  background: var(--vscode-editor-background);
}

.main-content {
  display: flex;
  flex: 1;
  overflow: hidden;
}
</style>
```

**useVscodeApi.ts - VS Code API 封装：**
```typescript
import { ref, onMounted, onUnmounted } from 'vue';

export function useVscodeApi() {
  const vscode = acquireVsCodeApi();
  const messageHandlers = ref<((message: any) => void)[]>([]);

  const postMessage = (message: any) => {
    vscode.postMessage(message);
  };

  const onMessage = (handler: (message: any) => void) => {
    messageHandlers.value.push(handler);
  };

  const handleMessage = (event: MessageEvent) => {
    const message = event.data;
    messageHandlers.value.forEach(handler => handler(message));
  };

  onMounted(() => {
    window.addEventListener('message', handleMessage);
  });

  onUnmounted(() => {
    window.removeEventListener('message', handleMessage);
  });

  return {
    postMessage,
    onMessage,
    vscode
  };
}
```

#### 3.2.2 核心类设计

**PublishFlowManager - 发布流程管理器：**
```typescript
export class PublishFlowManager {
  private config: PublishConfig;
  private configPath: string;
  
  constructor(workspacePath: string) {
    this.configPath = path.join(workspacePath, '.releasePlan', 'publish-config.json');
    this.config = this.loadConfig();
  }
  
  // 加载配置
  private loadConfig(): PublishConfig {
    if (fs.existsSync(this.configPath)) {
      const content = fs.readFileSync(this.configPath, 'utf-8');
      return JSON.parse(content);
    }
    return this.createDefaultConfig();
  }
  
  // 创建默认配置
  private createDefaultConfig(): PublishConfig {
    return {
      version: '1.0.0',
      createdAt: new Date().toISOString(),
      updatedAt: new Date().toISOString(),
      name: '默认发布配置',
      description: '',
      files: [],
      metadata: {
        totalFiles: 0,
        totalFolders: 0,
        totalSize: 0
      }
    };
  }
  
  // 添加文件
  addFile(sourcePath: string, targetPath: string): void {
    const fileName = path.basename(sourcePath);
    const stats = fs.statSync(sourcePath);
    
    const fileNode: FileNode = {
      id: uuidv4(),
      type: 'file',
      name: fileName,
      path: targetPath,
      sourcePath: sourcePath,
      size: stats.size,
      lastModified: stats.mtime.toISOString()
    };
    
    this.config.files.push(fileNode);
    this.updateMetadata();
  }
  
  // 添加文件夹
  addFolder(name: string, parentPath?: string): FileNode {
    const folderNode: FileNode = {
      id: uuidv4(),
      type: 'folder',
      name: name,
      path: parentPath ? path.join(parentPath, name) : name,
      children: []
    };
    
    if (parentPath) {
      const parent = this.findNodeByPath(parentPath);
      if (parent && parent.children) {
        parent.children.push(folderNode);
      }
    } else {
      this.config.files.push(folderNode);
    }
    
    this.updateMetadata();
    return folderNode;
  }
  
  // 查找节点
  private findNodeByPath(targetPath: string, nodes?: FileNode[]): FileNode | undefined {
    const searchNodes = nodes || this.config.files;
    
    for (const node of searchNodes) {
      if (node.path === targetPath) {
        return node;
      }
      if (node.children) {
        const found = this.findNodeByPath(targetPath, node.children);
        if (found) return found;
      }
    }
    return undefined;
  }
  
  // 更新元数据
  private updateMetadata(): void {
    let totalFiles = 0;
    let totalFolders = 0;
    let totalSize = 0;
    
    const traverse = (nodes: FileNode[]) => {
      for (const node of nodes) {
        if (node.type === 'file') {
          totalFiles++;
          totalSize += node.size || 0;
        } else {
          totalFolders++;
          if (node.children) {
            traverse(node.children);
          }
        }
      }
    };
    
    traverse(this.config.files);
    
    this.config.metadata = {
      totalFiles,
      totalFolders,
      totalSize
    };
  }
  
  // 保存配置
  async save(): Promise<void> {
    this.config.updatedAt = new Date().toISOString();
    
    const dir = path.dirname(this.configPath);
    if (!fs.existsSync(dir)) {
      await fs.promises.mkdir(dir, { recursive: true });
    }
    
    await fs.promises.writeFile(
      this.configPath,
      JSON.stringify(this.config, null, 2),
      'utf-8'
    );
  }
  
  // 获取配置
  getConfig(): PublishConfig {
    return this.config;
  }
  
  // 根据 ID 查找节点
  findNodeById(nodeId: string, nodes?: FileNode[]): FileNode | undefined {
    const searchNodes = nodes || this.config.files;
    
    for (const node of searchNodes) {
      if (node.id === nodeId) {
        return node;
      }
      if (node.children) {
        const found = this.findNodeById(nodeId, node.children);
        if (found) return found;
      }
    }
    return undefined;
  }
  
  // 根据 ID 移除节点
  removeNode(nodeId: string, nodes?: FileNode[]): boolean {
    const searchNodes = nodes || this.config.files;
    
    for (let i = 0; i < searchNodes.length; i++) {
      if (searchNodes[i].id === nodeId) {
        searchNodes.splice(i, 1);
        this.updateMetadata();
        return true;
      }
      if (searchNodes[i].children) {
        if (this.removeNode(nodeId, searchNodes[i].children)) {
          return true;
        }
      }
    }
    return false;
  }
  
  // 添加带子节点的文件夹（用于拖拽移动）
  addFolderWithChildren(folderNode: FileNode, parentPath?: string): void {
    const newPath = parentPath 
      ? path.join(parentPath, folderNode.name)
      : folderNode.name;
    
    const newFolder: FileNode = {
      ...folderNode,
      id: uuidv4(), // 生成新 ID
      path: newPath,
      children: folderNode.children ? 
        this.updateChildrenPaths(folderNode.children, newPath) : 
        []
    };
    
    if (parentPath) {
      const parent = this.findNodeByPath(parentPath);
      if (parent && parent.children) {
        parent.children.push(newFolder);
      }
    } else {
      this.config.files.push(newFolder);
    }
    
    this.updateMetadata();
  }
  
  // 更新子节点路径
  private updateChildrenPaths(children: FileNode[], parentPath: string): FileNode[] {
    return children.map(child => {
      const newPath = path.join(parentPath, child.name);
      return {
        ...child,
        id: uuidv4(), // 生成新 ID
        path: newPath,
        children: child.children ? 
          this.updateChildrenPaths(child.children, newPath) : 
          undefined
      };
    });
  }
}
```

```typescript
panel.webview.onDidReceiveMessage(

              const targetFilePath = targetPath

              manager.addFile(projectFile.sourcePath, targetFilePath);

                projectFile.sourcePath,

            panel.webview.postMessage({
              type: 'configUpdated',
              payload: manager.getConfig()
            });
          }
        }
        break;
        
      case 'moveInPublishList':
        // 在发布清单内部移动
        {
          const { itemId, targetPath } = message.payload;
          
          // 找到要移动的节点
          const nodeToMove = manager.findNodeById(itemId);
          
          if (nodeToMove) {
            // 检查是否会导致循环
            if (nodeToMove.type === 'folder' && targetPath?.startsWith(nodeToMove.path)) {
              vscode.window.showWarningMessage('不能将文件夹移动到自身或其子文件夹内');
              return;

            manager.removeNode(itemId);
            
            // 添加到新位置
            const newPath = targetPath 
              ? path.join(targetPath, nodeToMove.name)
              : nodeToMove.name;
            
            if (nodeToMove.type === 'file') {
              manager.addFile(nodeToMove.sourcePath, newPath);
            } else {
              manager.addFolderWithChildren(nodeToMove, targetPath);
            }
            
            // 更新 Webview
            panel.webview.postMessage({
              type: 'configUpdated',
              payload: manager.getConfig()
            });
          }
        }
```

```typescript
async function addFolderRecursive(
  folderPath: string,
  manager: PublishFlowManager,
  parentPath?: string
): Promise<void> {
  const folderName = path.basename(folderPath);
  const targetPath = parentPath 
    ? path.join(parentPath, folderName) 
    : folderName;
  
  const folderNode = manager.addFolder(folderName, parentPath);
  
  const entries = await fs.promises.readdir(folderPath, { withFileTypes: true });
  
  for (const entry of entries) {
    const fullPath = path.join(folderPath, entry.name);
    
    if (entry.isDirectory()) {
      await addFolderRecursive(fullPath, manager, targetPath);
    } else {
      manager.addFile(fullPath, path.join(targetPath, entry.name));

  }

async function addFolderRecursiveToTarget(

    } else {

function findFileInWorkspace(
  itemId: string,
  sourcePath: string,
  workspaceTree?: FileNode[]
): FileNode | undefined {
  // 实际实现时，workspaceTree 应该存储在全局或从 Extension 获取
  // 这里简化处理，直接构造 FileNode
  const stats = fs.statSync(sourcePath);
  
  return {
    id: itemId,
    type: stats.isDirectory() ? 'folder' : 'file',
    name: path.basename(sourcePath),
    path: sourcePath,
    sourcePath: sourcePath,
    size: stats.size,
    lastModified: stats.mtime.toISOString()
  };
}

// 扫描工作区文件树
async function scanWorkspace(workspacePath: string): Promise<FileNode[]> {
  const root: FileNode[] = [];
  
  async function scanDir(dirPath: string, relativePath: string): Promise<FileNode[]> {
    const nodes: FileNode[] = [];
    const entries = await fs.promises.readdir(dirPath, { withFileTypes: true });
    
    for (const entry of entries) {
      // 忽略隐藏文件和 node_modules
      if (entry.name.startsWith('.') || entry.name === 'node_modules') {
        continue;
      }
      
      const fullPath = path.join(dirPath, entry.name);
      const nodeRelativePath = path.join(relativePath, entry.name);
      
      if (entry.isDirectory()) {
        const children = await scanDir(fullPath, nodeRelativePath);
        nodes.push({
          id: uuidv4(),
          type: 'folder',
          name: entry.name,
          path: nodeRelativePath,
          children
        });
      } else {
        const stats = await fs.promises.stat(fullPath);
        nodes.push({
          id: uuidv4(),
          type: 'file',
          name: entry.name,
          path: nodeRelativePath,
          sourcePath: fullPath,
          size: stats.size,
          lastModified: stats.mtime.toISOString()
        });
      }
    }
    
    return nodes;
  }
  
  return scanDir(workspacePath, '');
}
```

## 6. 安全考虑

- 配置文件存储在本地，不上传到远程
- 敏感信息（如密码）不应存储在配置中
- 文件路径使用相对路径，避免泄露绝对路径
