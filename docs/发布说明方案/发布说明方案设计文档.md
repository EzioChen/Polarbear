# 发布说明方案设计文档

## 1. 功能概述

设计并实现一个发布说明编辑功能，采用填表形式，允许用户填写版本信息、发布内容、芯片范围、SDK版本等关键信息，并将配置保存为本地JSON文件。

## 2. 详细需求

### 2.1 入口设计
- 在侧边栏点击"发布说明"选项后，打开发布说明编辑界面
- 界面采用表单形式，包含多个填写区域

### 2.2 表单字段设计

#### 2.2.1 版本等级（必填）
- **类型**: 单选按钮组
- **选项**:
  - **A类**: 问题修复，客户必须修复
  - **B类**: 性能优化，客户可选择修复
  - **C类**: 功能增加，客户可选择修复
- **UI样式**: 三个卡片式选项，选中时高亮显示

#### 2.2.2 发布说明（必填）
- **类型**: Markdown 编辑器
- **格式支持**: 完整的 Markdown 语法
  - 标题（# ## ###）
  - 列表（有序/无序）
  - 粗体、斜体
  - 代码块
  - 链接
  - 表格
- **格式示例**:
  ```markdown
  ## 1、新增功能
  1. 增加兼容 AC707N 的自定义表盘图像转换；
  2. 分离图像转换工具作为独立模块库。

  ## 2、问题修复
  - 修复 XXX 问题；

  ## 3、性能优化
  - 优化 XXX 性能；
  ```
- **UI样式**: 
  - 分栏设计：左侧编辑区，右侧实时预览区
  - 支持全屏编辑模式
  - 提供常用 Markdown 工具栏（加粗、列表、代码块等）
  - 支持语法高亮

#### 2.2.3 使用芯片范围（必填）
- **类型**: 文本输入框
- **占位符**: "请输入支持的芯片型号，如：AC707N、AC608N"
- **UI样式**: 单行文本输入，支持标签式展示已输入的芯片型号

#### 2.2.4 对应固件SDK版本（必填）
- **类型**: 动态表格
- **表格列**:
  | 芯片 | 版本 | 备注 |
  |------|------|------|
  | AC707N | v2.1.0 | 稳定版本 |
  | AC608N | v1.5.3 | 测试版本 |
- **功能**:
  - 添加行
  - 删除行
  - 编辑单元格
- **UI样式**: 可编辑表格，带操作按钮

#### 2.2.5 SDK版本和APP版本（必填）
- **类型**: 双行文本输入
- **字段**:
  - SDK版本: 如 "v2.1.0"
  - APP版本: 如 "v1.0.5"
- **UI样式**: 两个并排或上下排列的输入框

#### 2.2.6 注意事项（可选）
- **类型**: 多行文本输入框
- **占位符**: "请输入需要注意的事项（可选）"
- **UI样式**: 可折叠的文本区域，默认收起

### 2.3 配置保存功能

**保存路径：**
```
workspace/
  ├─ .releasePlan/
  │   └─ release-notes.json
  └─ ...
```

**保存规则：**
- **自动保存**: 编辑时自动保存（防抖 500ms），防止数据丢失
- **手动保存**: 界面底部提供"保存"按钮，点击后显式保存并显示成功提示
- **取消按钮**: 点击取消关闭面板，不保存当前编辑（已自动保存的内容保留）
- 创建隐藏文件夹 `.releasePlan` 用于存储配置文件
- 文件命名: `release-notes.json`（固定文件名，与发布流程配置保持一致）
- 每次保存覆盖原文件，不保留历史版本
- **保存状态提示**: 在界面顶部或底部显示保存状态（已保存/保存中/未保存）

## 3. 数据结构设计

### 3.1 发布说明配置接口

```typescript
/**
 * 版本等级
 */
export type VersionLevel = 'A' | 'B' | 'C';

/**
 * 版本等级描述
 */
export interface VersionLevelInfo {
  level: VersionLevel;
  name: string;
  description: string;
  required: boolean;
}

/**
 * SDK版本信息
 */
export interface SdkVersionInfo {
  id: string;
  chip: string;
  version: string;
  remark: string;
}

/**
 * 发布说明配置
 */
export interface ReleaseNotesConfig {
  /** 配置版本 */
  version: string;
  /** 创建时间 */
  createdAt: string;
  /** 更新时间 */
  updatedAt: string;
  /** 版本等级 */
  versionLevel: VersionLevel;
  /** 发布说明内容 */
  releaseNotes: string;
  /** 使用芯片范围 */
  chipRange: string[];
  /** SDK版本列表 */
  sdkVersions: SdkVersionInfo[];
  /** SDK版本 */
  sdkVersion: string;
  /** APP版本 */
  appVersion: string;
  /** 注意事项 */
  remarks?: string;
  /** 元数据 */
  metadata: {
    author?: string;
    createdBy: string;
  };
}
```

### 5.2 核心类设计

```typescript
export class ReleaseNotesManager {
  private config: ReleaseNotesConfig;
  private configPath: string;
  private workspacePath: string;

  constructor(workspacePath: string) {
    this.workspacePath = workspacePath;
    this.configPath = path.join(workspacePath, '.releasePlan', 'release-notes.json');
    this.config = this.loadConfig();
  }

  /**
   * 加载配置
   */
  private loadConfig(): ReleaseNotesConfig {
    if (fs.existsSync(this.configPath)) {
      try {
        const content = fs.readFileSync(this.configPath, 'utf-8');
        return { ...defaultReleaseNotesConfig, ...JSON.parse(content) };
      } catch (error) {
        console.error('加载发布说明配置失败:', error);
      }
    }
    return this.createDefaultConfig();
  }

  /**
   * 创建默认配置
   */
  private createDefaultConfig(): ReleaseNotesConfig {
    return {
      ...defaultReleaseNotesConfig,
      createdAt: new Date().toISOString(),
      updatedAt: new Date().toISOString()
    };
  }

  /**
   * 获取配置
   */
  getConfig(): ReleaseNotesConfig {
    return this.config;
  }

  /**
   * 更新配置
   */
  updateConfig(updates: Partial<ReleaseNotesConfig>): void {
    this.config = {
      ...this.config,
      ...updates,
      updatedAt: new Date().toISOString()
    };
  }

  /**
   * 防抖保存（用于自动保存）
   */
  private debouncedSave: (() => void) | null = null;

  /**
   * 初始化防抖保存
   */
  initAutoSave(callback?: () => void): void {
    if (!this.debouncedSave) {
      this.debouncedSave = this.debounce(() => {
        this.save();
        callback?.();
      }, 500);
    }
  }

  /**
   * 防抖函数
   */
  private debounce(fn: () => void, delay: number): () => void {
    let timer: NodeJS.Timeout | null = null;
    return () => {
      if (timer) clearTimeout(timer);
      timer = setTimeout(() => {
        fn();
      }, delay);
    };
  }

  /**
   * 触发自动保存
   */
  autoSave(callback?: () => void): void {
    this.initAutoSave(callback);
    this.debouncedSave?.();
  }

  /**
   * 保存配置
   */
  async save(): Promise<void> {
    const dir = path.dirname(this.configPath);
    if (!fs.existsSync(dir)) {
      fs.mkdirSync(dir, { recursive: true });
    }

    // 保存到 release-notes.json（固定文件名，与发布流程配置保持一致）
    fs.writeFileSync(
      this.configPath,
      JSON.stringify(this.config, null, 2),
      'utf-8'
    );
  }

  /**
   * 添加SDK版本
   */
  addSdkVersion(info: Omit<SdkVersionInfo, 'id'>): void {
    const newVersion: SdkVersionInfo = {
      ...info,
      id: `sdk-${Date.now()}`
    };
    this.config.sdkVersions.push(newVersion);
    this.config.updatedAt = new Date().toISOString();
  }

  /**
   * 删除SDK版本
   */
  removeSdkVersion(id: string): void {
    this.config.sdkVersions = this.config.sdkVersions.filter(v => v.id !== id);
    this.config.updatedAt = new Date().toISOString();
  }

  /**
   * 更新SDK版本
   */
  updateSdkVersion(id: string, updates: Partial<SdkVersionInfo>): void {
    const index = this.config.sdkVersions.findIndex(v => v.id === id);
    if (index !== -1) {
      this.config.sdkVersions[index] = {
        ...this.config.sdkVersions[index],
        ...updates
      };
      this.config.updatedAt = new Date().toISOString();
    }
  }
}
```

<template>

    <!-- SDK和APP版本 -->
    <FormSection title="SDK版本和APP版本" required>
      <div class="version-row">
        <VersionInput

        />
      </div>
    </FormSection>

    <!-- 注意事项 -->
    <FormSection title="注意事项" collapsible>
      <textarea
        v-model="form.remarks"
        class="remarks-textarea"
        placeholder="请输入需要注意的事项（可选）"
        rows="4"
      />
    </FormSection>

    <!-- 保存状态栏 -->
    <div class="save-status-bar" :class="saveStatus">
      <span v-if="saveStatus === 'saving'">保存中...</span>
      <span v-else-if="saveStatus === 'saved'">已保存</span>
      <span v-else>未保存</span>
    </div>

    <!-- 操作按钮 -->
    <div class="form-actions">
      <button class="btn-secondary" @click="handleCancel">取消</button>
      <button class="btn-primary" @click="handleSave">保存</button>
    </div>
  </div>

<script setup lang="ts">
import { reactive, ref, watch } from 'vue';
import type { ReleaseNotesConfig, VersionLevelInfo } from '../types';
import { versionLevelOptions, defaultTemplate } from '../constants';
import FormSection from './FormSection.vue';
import VersionLevelSelector from './VersionLevelSelector.vue';
import MarkdownEditor from './MarkdownEditor.vue';
import ChipRangeInput from './ChipRangeInput.vue';
import SdkVersionTable from './SdkVersionTable.vue';
import VersionInput from './VersionInput.vue';

const props = defineProps<{
  initialConfig?: Partial<ReleaseNotesConfig>;
}>;

const emit = defineEmits<{
  save: [config: ReleaseNotesConfig];
  cancel: [];
  autoSave: [config: ReleaseNotesConfig];
}>();

// 保存状态：'unsaved' | 'saving' | 'saved'
const saveStatus = ref<'unsaved' | 'saving' | 'saved'>('saved');

const form = reactive<ReleaseNotesConfig>({
  version: '1.0.0',
  createdAt: new Date().toISOString(),
  updatedAt: new Date().toISOString(),
  versionLevel: 'C',
  releaseNotes: defaultTemplate,
  chipRange: [],
  sdkVersions: [],
  sdkVersion: '',
  appVersion: '',
  remarks: '',
  metadata: {
    createdBy: 'polarbear-extension'
  },
  ...props.initialConfig
});

// 监听表单变化，触发自动保存
watch(
  () => ({ ...form }),
  () => {
    saveStatus.value = 'unsaved';
    debouncedAutoSave();
  },
  { deep: true }
);

// 防抖自动保存（500ms）
let autoSaveTimer: NodeJS.Timeout | null = null;
const debouncedAutoSave = () => {
  if (autoSaveTimer) clearTimeout(autoSaveTimer);
  autoSaveTimer = setTimeout(() => {
    saveStatus.value = 'saving';
    emit('autoSave', { ...form });
    // 模拟保存完成
    setTimeout(() => {
      saveStatus.value = 'saved';
    }, 300);
  }, 500);
};

const addSdkVersion = () => {
  form.sdkVersions.push({
    id: `sdk-${Date.now()}`,
    chip: '',
    version: '',
    remark: ''
  });
};

const removeSdkVersion = (id: string) => {
  form.sdkVersions = form.sdkVersions.filter(v => v.id !== id);
};

const updateSdkVersion = (id: string, updates: any) => {
  const index = form.sdkVersions.findIndex(v => v.id === id);
  if (index !== -1) {
    form.sdkVersions[index] = { ...form.sdkVersions[index], ...updates };
  }
};

const handleSave = () => {
  emit('save', { ...form });
  saveStatus.value = 'saved';
};

const handleCancel = () => {
  emit('cancel');
};
</script>
